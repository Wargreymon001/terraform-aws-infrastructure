# Terraform AWS Infrastructure

Proyecto personal donde implemento una infraestructura cloud básica en AWS utilizando Infrastructure as Code con Terraform.

## Tecnologías utilizadas
- Terraform
- AWS
- Git
- GitHub
- Linux

## Arquitectura

La infraestructura desplegada incluye:

- VPC
- Subnet pública
- Internet Gateway
- Route Table
- Security Group
- Instancia EC2
